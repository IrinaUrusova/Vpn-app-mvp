# Что нужно подготовить от Ирочки

1. Финальное **название приложения**.
2. Логотип в 2 форматах: SVG + PNG (1024x1024).
3. Brand-палитра (подтвердить):
   - `#6FEAF2`, `#8A5CFF`, `#EADFA8`, `#0A1026`
4. Тексты UI:
   - Главная кнопка: «Подключить» / «Отключить»
   - Recovery: «Починить подключение»
5. Юридические ссылки:
   - Privacy Policy URL
   - Terms URL
6. Продуктовые параметры:
   - Лимит перевыпуска (предлагаю 1/10 мин, 5/сутки)
   - Максимум устройств на клиента
7. Доступы для интеграции:
   - Тестовая панель (x-ui/3x-ui) URL
   - API credentials (без отправки в чат, лучше через защищённый канал)
